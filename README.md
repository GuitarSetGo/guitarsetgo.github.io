### ⚡ Level Up Your URL

Did you know? Usually, GitHub Pages URLs look like `https://<user>.github.io/<project-name>`.

However, this repository is special. I named it **`guitarsetgo.github.io`**.

Because of this exact naming convention, GitHub automatically treats it as my main site and publishes it to the root domain:

👉 **https://guitarsetgo.github.io/**

It's the cleanest way to host your personal page on GitHub! 🏠
